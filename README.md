# 🏭 Quality Control Telegram Bot + Dashboard

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://python.org)
[![Telegram](https://img.shields.io/badge/Telegram-Bot-blue.svg)](https://core.telegram.org/bots)
[![Dash](https://img.shields.io/badge/Dash-Plotly-green.svg)](https://dash.plotly.com)

> Telegram-бот для автоматизации контроля качества товаров + интерактивный дашборд для визуализации.

---

## 🎯 Цель проекта

Автоматизировать процесс сбора и анализа данных о контроле качества товаров. Сотрудники ОТК вносят результаты проверок через Telegram-бота, данные мгновенно сохраняются в базу и отображаются на веб-дашборде.

---

## ✨ Возможности

### 🤖 Telegram-бот
- Регистрация новых проверок качества
- Фиксация результатов (годен/брак)
- Выбор категории дефекта
- Просмотр статистики качества
- Просмотр своих проверок

### 📊 Дашборд
- Процент годных изделий (Gauge-индикатор)
- Распределение дефектов по категориям
- Динамика результатов проверок
- Таблица с последними проверками
- Автообновление каждые 5 секунд
- Фильтры по дате и результату

## 📥 Установка и запуск

### 1. Клонирование репозитория

`git clone https://github.com/ВАШ_ЛОГИН/quality-control-bot.git`

`cd quality-control-bot`

### 2. Создание виртуального окружения

`python -m venv venv`

Для Mac/Linux: `source venv/bin/activate`

Для Windows: `venv\Scripts\activate`

### 3. Установка зависимостей

`pip install -r requirements.txt`

### 4. Настройка бота

1. Найдите в Telegram **@BotFather**
2. Отправьте команду `/newbot`
3. Получите токен
4. Вставьте токен в файл `bot.py` (строка `BOT_TOKEN = "YOUR_TOKEN_HERE"`)

### 5. Запуск

**Терминал 1 — Telegram-бот:** `python bot.py`

**Терминал 2 — Дашборд:** `python dashboard.py`

### 6. Открыть в браузере

`http://127.0.0.1:8050`

## 📖 Как пользоваться

### Команды бота

| Команда | Описание |
|---------|----------|
| /start | Главное меню |

### Сценарий использования

1. Отправьте /start боту
2. Нажмите "Новая проверка"
3. Введите ID партии и наименование товара
4. Выберите результат (Годен/Брак)
5. При браке — выберите категорию дефекта
6. Готово! Данные появились на дашборде
